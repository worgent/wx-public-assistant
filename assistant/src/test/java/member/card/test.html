人们常用 “少时了了，大未必佳”来形容一些神童级人物的发展路径的。一个不容忽视的现象是：在漫漫人生途中，为数不少的人从踌躇满志逐渐沦为凡俗平庸。童年的时候，人人都有不俗的梦想：我长大了要当警察、我长大了要当科学家、我长大了要当官……不少人的梦想一直能延续到大学毕业，在大学毕业踌躇满志地走向社会的时候，他们的梦想还热乎着呢。早在2006年我就在一次演讲中说：人和人的差距不是小学、中学乃至大学期间拉开的，而是在大学毕业之后拉开的。分水岭是：走向社会，你是否依旧怀揣着你的梦想。据我多年观察、跟踪、研究发现，梦想是这样一步步被侵蚀的。亲，你们看，是也不是？

 

 

 

第一步：梦想遭现实侵袭

梦想很丰满，现实很骨感。梦想在现实面前很容易显得苍白无力，美好的梦想总会被财迷油盐酱醋茶的现实侵袭，美好的梦想常常遭那些也曾经踌躇满志却在现实中梦想碎了一地的学长学姐冠以幼稚的称号讥笑，美好的梦想总会遇到曲曲折折的困难。梦想很重要但比较遥远，现实的困难却就在眼前。独立谋生这一关，就足以把多数人的梦想碾碎。是也不是？

 

 

 

第二步，缺乏梦想的人生必被惯性支配

人们的行为由两股力量支配，一股是梦想的力量，一股是惯性的力量。此二者是此消彼长的关系，梦想的力量大些，随波逐流的习惯的力量就小些，梦想的力量小了，习惯的量就大了。当一个人习惯了随波逐流和跟着感觉走的时候，就开始逐渐进入沦为凡俗轨道了。是也不是？

 

 

 

第三步，惯性的行为不会有超乎想象的结果

天上不会掉馅饼，梦想也不会不努力自己实现的。惯性的行为不会带来意外的收获，相反，可能因为外部坏境的变化，连原来的收获都会打折，从差强人意逐渐沦为勉强凑合。是也不是？

 

 

 

第四步，残酷的现实让你不敢有梦想

勉强凑合的结果变成现实，残酷地让人不敢再奢望更大的梦想，连想都不敢想！至此，一个劣性循环就此闭合，此后不断迭代，三次迭代就足以把一个踌躇满志的人沦为平庸凡俗。是也不是？

 

 

 

哈佛大学曾经做过一项抽样调查显示：只有大约3%的人有长远的目标，大约10%的人有短期清晰的目标，大约60%的人目标模糊，还有大约27%的人没有目标。研究这对这些样本进行长达的跟踪，结果发现：20年以后，没有目标的群体成员绝大部分生活在社会底层，整体抱怨社会不公；而目标比较模糊的群体则多为工薪层或中上层，生活比较稳定；有短期清晰目标的10%的群体，多为社会精英和专业人士；而剩下3%的有长期目标的人，最后成为社会顶尖人物。所以，我们今天的生活状态并不由今天决定，而是我们过去生活目标的结果！明天生活在什么状态，则要看我们今天想成为谁。

亲，如果你的梦想已经被束之高阁、蛛网尘封，这篇文章意在提醒你，该拿出来擦擦灰尘了。
